
int convertir_en_secondes(Temps t)                                           		FAIT 
	-> Transforme une heure/minute/seconde en un seul nombre de secondes.
utile pour
	-> calculer le temps d’attente d’un patient
	-> comparer l’heure actuelle et l’heure d’arrivée




void mettre_a_jour_humeur_patient(Patient *p, Temps temps_actuel)			FAIT 
	->Met à jour l’humeur du patient selon le temps qu’il a attendu.

logique : 

	° moins de 3 min → `VERT`
	° à partir de 3 min → `ORANGE`
	° à partir de 5 min → `ROUGE`




float calculer_montant_patient(Patient *p)						FAIT 

	->Calculer combien le patient doit payer.

	° `SATISFAIT` → paie tout
	° `MECONTENT` → paie la moitié
	° `FURIEUX` → paie 0


Cette fonction calcule seulement, elle ne modifie pas l’argent du cabinet.




void verifier_gants(Dentiste *d, Patient *p)						FAIT 

	-> Vérifie si le dentiste porte des gants.

	°pas de gants → le patient devient rouge/furieux et part




void verifier_proprete_gants(Dentiste *d, Patient *p)					FAIT 

	-> Vérifie si les gants sont propres.


	° gants sales → patient furieux, il part




void verifier_proprete_ustensile(Dentiste *d, Patient *p)				FAIT 


	-> Vérifie si l’ustensile manipulé est propre.

	° ustensile sale → patient furieux, il part



void verifier_ustensile_pathologie(Dentiste *d, Patient *p)				FAIT  

	->Vérifie si l’ustensile tenu correspond à la pathologie du patient.

	° si la pathologie demande un certain ustensile et que ce n’est pas le bon → patient part



void gerer_patient(Patient *p, Dentiste *d, Temps temps_actuel)				FAIT 


	-> Fonction centrale de gestion du patient.

	° vérifie que le patient est encore là
	° met à jour son humeur
	° vérifie les gants
	° vérifie la propreté des gants
	° vérifie la propreté de l’ustensile
	° vérifie l’ustensile / la pathologie




int peut_soigner_patient(Dentiste *d, Patient *p)					FAIT 


	-> Dit si le soin est possible ou non.


	° pas de patient dans le fauteuil → non
	° pas de gants → non
	° main occupée → non
	° sinon → oui

void prendre_ustensile(Dentiste *d, int numero_ustensile)


	-> Fait prendre un ustensile au dentiste.

	° si la main est déjà occupée → rien
* sinon :

	° il prend l’ustensile
 	° main = 1`
	° ustensile_en_main = numero_ustensile`
	°si pas de gants → l’ustensile devient sale
	° sinon → l’ustensile est propre

LA fonction exécute juste la prise.





void deposer_ustensile_sur_plateau(Dentiste *d, Patient *p)

	-> Dépose l’ustensile porté sur le plateau du patient.

	°  si main vide → rien
	° si l’ustensile est sale → rien
sinon :

	° on met l’ustensile sur le plateau
	° la main redevient vide
	° ustensile_en_main = 0`


// faire tableau d ustensile qui sera le plateau
